Detailed Description

Settings message (for CommunicationService)

This class represents a communication message to be sent to FlySens when accessing the settings of the system. The message can be used to read or write any settings of the FlySens device. Please refer to the protocol description for the full details.

Parameters
inSendStringThe settings string is either just the setting name (e.g. "QNH_Pa") to read a setting or it is a assignment string setting=value (e.g. "QNH_Pa=101325") to write a setting.
inExpectResponseTRUE if a response from FlySens is expected. If FALSE no response will be sent. A read setting command should have set paramter to true
