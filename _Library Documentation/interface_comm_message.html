Detailed Description

Generic Message class for CommunicationService.

This implementation is the base implementatoin for a message. It may be derived for specific messages

- (void) getMessageData: (NSMutableData*)  messageData

Returns the message data.

This function adds the message data to the NSMutableData object. The function adds the data such as it should be sent, but without taking care of the message splitting. This will be done by the CommunicationService upon sending. This function is called from the CommunicationService to get the data to be sent

Parameters
messageDataReference to NSMutableData to be filled with the data to be sent
